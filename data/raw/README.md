# Dataset Setup

This project uses the Enron Email Dataset.

Due to GitHub size limitations, the dataset is not included in this repository.

## Download Dataset

Download the dataset from:

https://www.cs.cmu.edu/~enron/

Or the cleaned CSV version used for this project:

https://www.kaggle.com/datasets/wcukierski/enron-email-dataset

## Setup

1. Download the dataset
2. Extract it
3. Place the CSV file in the following directory:

data/raw/

The final structure should look like:

data/
└── raw/
└── sample_email.csv
└── enron_sample.csv
